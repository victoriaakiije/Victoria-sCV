About Me


I'm a graphics designer, front and back end developer based in Lagos state, Nigeria.
I started with graphics design and advanced to web application development in 2017 during my NYSC at an Information and Communication Technology firm. At the time, Corel Draw,Photoshop, HTML, CSS, were my tools of development. More recently, I discovered my love for JavaScript and Bootstrap as I took courses in Udacity and w3schools as ways to improve myself and learn more skills.

My HTML and CSS skills are very good and I'm conversant with the use of JavaScript and PHP. I am also very familiar with web responsive tools such as Bootstrap.
I am currently working on learning a number of front end frameworks to improve my front end JavaScript skills. I have also learned to work with certain frameworks like Node JS and Express JS. I have experience working in a fintech based on my internship.
